# CmdLine_OrgChart_12.1

## Title: The Command Line OrgChart App
<br>

## Description:
This is a command-line application to manage an employee database. It uses Node.js, Inquirer, and MySQL.
<br><br>

## Usage (type in terminal):
npm i
<br>
node server.js
